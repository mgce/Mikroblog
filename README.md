# Mikroblog

Social App from category „Social
News”. The main thrust is sharing informations beetwen users, by
posts. A very usefull fetures are tags. They
allow user to sign their posts. It helps to search posts of
intresting topic for us. 


Users can:
- Add, Edit and Delete Posts,
- Add tags to posts,
- Follow chosen tags, all post with followed tags are showing on page „My following tags”
- User Profile, where are all posts writed by user


For this project I have used ASP.NET
4.5 MVC, Entity Framework and jQuery.
